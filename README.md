# RequestFlow

A full-stack Role-Based Request & Approval Management System built using the MERN stack.

---

## Tech Stack

### Frontend
- React
- Vite
- Redux Toolkit
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB

## Clone the Repository

```bash
git clone https://github.com/Tharunecious/RequestFlow.git
cd repository-name
```

---


## How to Open the Project

1. Start the **backend server** first:
   ```bash
   cd backend
   npm install
   npm run dev
   ```

2. Open a **new terminal** and start the **frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. Open browser and go to:

```txt
http://localhost:5173
```
